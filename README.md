# SESD WORKSHOP-2

---

## CLI Tool

A Command Line Interface (CLI) application built using **Node.js**, **TypeScript**, and **Commander.js**.
The project follows **Object-Oriented Programming (OOP)** principles and integrates external APIs.

---

## Features

* TypeScript based CLI application
* Commander.js for command handling
* Object-Oriented project structure
* Multiple custom CLI commands
* External API integrations
* File system operations
* Colored terminal output using Chalk

---

## Technologies Used

* Node.js
* TypeScript
* Commander.js
* Axios
* Chalk@4

---

## APIs Used

* GitHub API
* ZenQuotes API
* Advice Slip API
* WeatherAPI

---

* greet
* add
* subtract
* multiply
* divide

* quote
